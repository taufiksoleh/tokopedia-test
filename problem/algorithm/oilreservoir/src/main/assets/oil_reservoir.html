Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much oil it is able to collect.




The above elevation map is represented by array [0,1,0,2,1,0,1,3,2,1,2,1]. In this case, 6 units of oil (yellow section) are being collected.


Example

    Input:

      [0,1,0,2,1,0,1,3,2,1,2,1]
    


    Output:

      6
